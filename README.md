# 🌸 MY Dreams - Website Baju Custom dengan AI

Website pemesanan baju custom dengan fitur **AI Design Studio** yang canggih.

## Fitur Utama
- Desain baju menggunakan AI (text-to-design)
- Background flower print dinamis + animasi petal
- Live Chat dengan desainer
- Pilihan jenis kain & kriteria model
- Responsive design (mobile friendly)

## Teknologi
- HTML5 + Tailwind CSS
- JavaScript vanilla
- Animasi CSS murni

## Cara Menjalankan
1. Download atau clone repository ini
2. Buka file `index.html` di browser
3. Atau deploy ke Vercel/Netlify (rekomendasi)
